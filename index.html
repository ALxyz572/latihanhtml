Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web. HTML berfungsi untuk:

  • Membuat struktur halaman (seperti heading, paragraf, gambar, link, dll).
  • Memberikan makna dan fungsi pada konten web.
  • Mengatur tata letak dasar dari halaman web.

Contoh Struktur HTML Dasar:

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Halo, Dunia!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
</body>
</html>

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan halaman web. CSS berfungsi untuk:

  • Memberikan warna, font, dan ukuran elemen HTML.
  • Mengatur posisi dan layout elemen di halaman.
  • Memberikan efek animasi dan transisi pada halaman web.

Contoh Penggunaan CSS Dasar:

body {
    background-color: #f9f9f9;
    font-family: Arial, sans-serif;
}

h1 {
    color: #0077cc;
    text-align: center;
}

p {
    color: #555;
    font-size: 16px;
}

Cara Menghubungkan HTML dengan CSS

Ada tiga cara utama untuk menggunakan CSS dalam HTML:

  1. Inline CSS: Menulis CSS langsung pada tag HTML menggunakan atribut style.
  2. Internal CSS: Menulis CSS dalam tag <style> di bagian <head>.
  3. External CSS: Menyambungkan file CSS terpisah menggunakan tag <link>.

Contoh Inline CSS:

<h1 style="color: blue;">Halo Dunia!</h1>

Contoh Internal CSS:

<!DOCTYPE html>
<html>
<head>
    <style>
        p {
            color: green;
        }
    </style>
</head>
<body>
    <p>Ini paragraf hijau.</p>
</body>
</html>

Contoh External CSS:

<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Judul Dengan External CSS</h1>
</body>
</html>
